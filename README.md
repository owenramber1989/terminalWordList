## Get started!

```zsh
git clone https://github.com/owenramber1989/terminalWordList.git;
cd terminalWordList;
make
```

## Configure

You can configure your database info in the file voc.conf. The default options is

- user: root
- password:
- host: localhost
- database: vocabulary

## Usage

1. Get the help message

`./voc -h`
